# **App Name**: CampusCycle

## Core Features:

- College-Verified Authentication: Secure authentication system using Firebase Auth with college email verification, ensuring only students within the campus can access the platform.
- Marketplace Listings: Allow sellers to post items to sell in the platform with relevant information such as description, pricing, images and categories
- Real-time In-App Chat: Enable real-time communication between buyers and sellers using Firebase Firestore, facilitating quick and efficient transactions.
- Wishlist Functionality: Allow users to save items to a wishlist for future reference.
- Rating and Review System: Enable buyers to rate and review sellers after a transaction, building trust and transparency within the campus community.
- AI-Powered Listing Assistant: Leverage generative AI tool to suggest optimal pricing and categorize the product listing using provided images and description. This tool ensures accuracy and attracts more potential buyers.
- Sustainability Tracking: Implement a Green Impact counter to track and promote the positive environmental impact of buying and selling used items within the campus community.

## Style Guidelines:

- Primary color: Vibrant blue (#3B82F6), inspired by the concepts of trust and connection to the campus.
- Background color: Light blue (#E0F7FA), providing a clean and fresh base.
- Accent color: Green (#34D399), reflecting the sustainable and eco-friendly aspects of the platform.
- Body and headline font: 'Inter' sans-serif for a clean, modern, and readable design.  'Poppins' for headlines for a precise feel. Note: currently only Google Fonts are supported.
- Use consistent, minimalist icons throughout the platform to enhance user experience.
- Card-based layout with rounded corners and soft shadows to create a modern and visually appealing interface.
- Incorporate micro-animations and smooth transitions to provide a seamless and engaging user experience.
